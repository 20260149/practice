[정리]
Q1. 원형 vs AABB: 모서리 부분에서 어떤 차이?
=> 사각형의 모서리가 닿지 않아도 원의 범위가 겹치면 충돌로 판단됨.

Q2. AABB vs OBB: 회전 시 어떤 차이?
=> AABB는 축에 평행한 사각형이라 오브젝트가 회전해도 충돌 박스는 회전하지 않음.
OBB는 오브젝트 회전을 따라 함께 회전함.

Q3. 어떤 방식이 내 게임에 더 적합할까?
=> 현재 만든 프로그램처럼 회전하는 적이 있는 경우는 OBB가 가장 적합함.

Q4. AI에게 물어본 것 중 흥미로웠던 답변?
=>OBB 충돌 구현하는 점에서 OBB 충돌은 단순한 사각형 비교보다 훨씬 정확하고,
회전하는 오브젝트에도 잘 맞는다는 점이 인상적이었다.
